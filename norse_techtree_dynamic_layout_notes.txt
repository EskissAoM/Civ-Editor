Norse dynamic right-side TechTree layout notes

This build uses the shared dynamic lane planner for Norse TechTree output.

Rules applied:
- Building/lane order is preserved across ages.
- Each building reserves a start column globally across Archaic/Classical/Heroic/Mythic.
- A new building lane starts only after the previous lane has enough width for all active nodes.
- Parent/child lines use one column whenever possible.
- Parent-line protection is based on actual active parent/child lines, not hardcoded one-off cases.
- Multi-age upgrade chains keep their column across the ages where they appear.
- One-age parent lines only protect their column in that age.
- A node can appear from different buildings, but duplicate nodes from the same building lane are suppressed.
- Units prefer row 0 when the parent building is not present in the same age; if the parent building is present and the unit is childless, row 1 may be used to keep the lane compact.
- ArchaicAge keeps its row-1 preference for non-root nodes where possible.
- WallConnector upgrades can share the WallConnector column.
- Temple Omniscience can share the Temple column.
- Thor Dwarven Armory bonus replaces the Armory lane with DwarvenArmory and keeps normal armory upgrades plus DwarvenWeapons, MeteoricIronArmor, and DragonscaleShields.

Latest row-packing adjustment:
- Direct childless units from a building that appears in the current age now prefer row 1 in their own columns.
- If a childless unit and childless tech compete for the same column under that building, the unit uses row 0 and the tech uses row 1.
- This is column-specific, so lanes like Longhouse can keep Hirdman/ThrowingAxeman/Berserk on row 1 while lanes like HillFort can still split units/techs in a shared column cleanly.
